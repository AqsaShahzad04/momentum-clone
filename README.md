# Momentum Clone — New Tab Dashboard Extension

A clone of the popular Momentum Chrome extension. Replaces the default new tab page with a focused dashboard showing the current time, a greeting, and a daily inspirational quote.

---

## Features

- Displays current time and a personalised greeting
- Daily focus prompt to set your intention for the day
- Clean, distraction-free full-screen background
- Runs as a Chrome browser extension

---

## Tech Stack

HTML · CSS · JavaScript · Chrome Extension API · Vite

---

## Getting Started

```bash
npm install
npm run build
```

Then load the `dist/` folder as an unpacked extension in `chrome://extensions`.
